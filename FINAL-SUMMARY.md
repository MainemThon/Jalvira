## 🐛 Troubleshooting

### Issue: Server won't start
**Solution:**
```bash
npm install
npm run dev
```

### Issue: Build fails
**Solution:**
```bash
node --version  # Check Node.js 18+
rm -rf node_modules package-lock.json
npm install
npm run build
```

### Issue: Animations not smooth
**Solution:**
- Check browser (use Chrome/Firefox latest)
- Reduce number of simultaneous animations
- See ANIMATION-GUIDE.md

### Issue: Logo not showing
**Solution:**
- Add `public/jalvira-logo.png`
- Or update path in Navbar.jsx
- See LOGO-GUIDE.md
